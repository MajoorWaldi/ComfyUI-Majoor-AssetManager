# Majoor Assets Manager - Hotkeys & Keyboard Shortcuts Guide

## Overview
This guide covers all active keyboard shortcuts for the Majoor Assets Manager.

## Global / Panel Hotkeys

| Shortcut | Action | Scope |
|----------|--------|-------|
| **Ctrl+S** / **Cmd+S** | Trigger Index Scan | Global (Panel Focused) |
| **Ctrl+F** / **Ctrl+K** | Focus Search Input | Global |
| **Ctrl+H** | Clear Search Input | Global |
| **D** | Toggle Sidebar (Details) | Grid / Panel |

## Grid View Hotkeys

These shortcuts apply when the asset grid has focus.

### Navigation & Selection
| Shortcut | Action |
|----------|--------|
| **Arrow Keys** | Navigate selection |
| **Enter** / **Space** | Open Viewer |
| **Ctrl+A** | Select All |
| **Ctrl+D** | Deselect All |
| **Ctrl+Click** | Toggle Selection |
| **Shift+Click** | Range Selection |

### Organization & Rating
| Shortcut | Action |
|----------|--------|
| **1** - **5** | Set Rating (1-5 stars) |
| **0** | Reset Rating |
| **T** | Edit Tags |
| **B** | Add to Collection (Bookmark) |
| **Shift+B** | Remove from Collection |

### File Operations
| Shortcut | Action |
|----------|--------|
| **F2** | Rename File |
| **Delete** | Delete File |
| **Ctrl+Shift+C** | Copy File Path |
| **Ctrl+Shift+E** | Open in Explorer/Finder |

---

## Viewer Hotkeys

These shortcuts apply when the Viewer overlay is open.

### General
| Shortcut | Action |
|----------|--------|
| **Esc** | Close Viewer |
| **F** | Toggle Fullscreen |
| **D** | Toggle Info Panel (Generation Data) |
| **Space** | Play/Pause Video |

### Navigation
| Shortcut | Action | Notes |
|----------|--------|-------|
| **Left** / **Right** | Previous / Next Asset | Default behavior |
| **Left** / **Right** | Step Frame (-1/+1) | **Only when Video Player bar is focused** |
| **Mouse Wheel** | Zoom In/Out | |
| **Click+Drag** | Pan Image | |

### Tools & Analysis
| Shortcut | Action |
|----------|--------|
| **I** | Toggle Pixel Probe |
| **C** | Copy Probed Color (Hex) |
| **L** | Toggle Loupe |
| **Z** | Toggle Zebra (Exposure) |
| **G** | Cycle Grid Overlays |
| **Alt+1** | Toggle 1:1 Pixel View |
| **+** / **-** | Zoom In / Out |
